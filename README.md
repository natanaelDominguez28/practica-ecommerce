# Proyecto Ecommerce
En este proyecto integrador combinamos los conocimientos del curso React + Redux con TypeScript dictado por Humberto Rivero para Digital House.

## Tecnologías utilizadas
+ React.js
+ React Credit Cards 2
+ React Router Dom
+ Sonner
+ React Query

## Instalación
# Clona el repositorio
git clone https://github.com/natanaelDominguez28/practica-ecommerce.git

# Instala las dependencias
npm install

# Corre la aplicación
npm run dev
